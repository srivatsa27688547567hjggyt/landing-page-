# Echelon International Landing Page

A high-end, cinematic landing page built with Next.js 14 and Tailwind CSS.

## 📁 Repository Structure

- `echelon-site/`: The main Next.js application.
- `public/`: Root-level public assets (ignored by git, only for local reference).
- `.agent/`: AI configuration and scripts (ignored by git).

## 🚀 Getting Started

To run the development server:

```bash
cd echelon-site
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🌐 Deployment

This site is optimized for deployment on **Vercel**. 
When importing the project to Vercel:
1. Set the **Root Directory** to `echelon-site`.
2. Vercel will automatically detect the Next.js framework.
